google hack技巧总结

发表于 2018-09-09 | 分类于 hacker | 阅读次数:

google hacker 技巧总结,更多google搜索 “google hack” filetype:pdf

黄金法则

1. Google查询不区分大小写。
2. Google通配符*仅代表搜索词组中的一个词。
3. Google省略停用词。

操作符

1. and(+) 在查询中包含多个关键字，google默认
2. not(-) 在查询中不包括某些关键字
3. or(|) 在查询中搜索一个或另一个关键字

高级操作符

1. intitle:"index of" “private” #返回标题中包含index of且在网页任何地方包含单词private的网页
2. Allintitle:"index of" "private" #返回标题中存在index of,private 的网页
3. inurl:sign.php #返回url中存在sign.php的网页
4. Allinurl:admin login.php #返回url中存在admin,login.php的网页
5. site:baidu.com #查找位于baidu.com上的网页
6. site:baidu.com filetype:pdf #filetype搜索指定类型的文件
7. site:freebuf.com link:www.baidu.com #link 搜索链接到一个网站的url
8. site:mytianya.me inanchor:linux #inanchor 在链接的描述文本中找关键字

linux 定时任务crontab命令详解

发表于 2018-06-03 | 分类于 linux命令 | 阅读次数:

  cron是linux下定时执行工具，可以在无需人工干预的情况下进行作业。cron服务提供了crontab命令来设置cron服务。

crontab命令

常见命令

1. crontab -l #查看当前用户下定时任务
2. crontab -r #删除当前用户下定时任务
3. crontab -e #编辑当前用户下定时任务

配置语法

使用crontab -e编辑当前用户下定时任务，会用默认编辑器配置定时任务,语法如下图:

1. 前五个分别是分·时·天·月·星期,最后一个需要执行的脚本命令
2. eg: * */2 * * * command  # / 表示频率，这里表示每隔2小时执行一次脚本
3. eg: 1,15 * * * * command # , 表示并列，这里表示每小时的1分钟，15分钟各执行一次
4. eg: * 8-15 * * * command # - 表示范围，这是表示每天8-15点每分钟执行一次
